# My-Portfolio

<h2> ✧ Descrição </h2>

<b>1.</b> Para visualizar o projeto localmente, siga os passos abaixo:<br><br>

<b>•</b> Clique no botão verde "<code>Code</code>" na parte superior do repositório.<br>
<b>•</b> Selecione "<code>Download ZIP</code>" para baixar os arquivos do projeto.<br>
<b>•</b> Extraia o conteúdo do arquivo ZIP em uma pasta de sua preferência.<br>
<b>•</b> Abra o arquivo <code>index.html</code> no seu navegador (basta dar dois cliques ou clicar com o botão direito e selecionar "Abrir com").<br><br>
